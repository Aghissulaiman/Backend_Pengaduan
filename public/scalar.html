📋 Pengaduan Provinsi API

Dokumentasi API untuk sistem pengaduan masyarakat tingkat provinsi

Base URL: http://localhost:8080/api

👥 Role dalam Sistem

Role Akses Tugas
👤 User (Warga) Submit & cek status Mengadu, melihat status
🔍 Investigator Cek lapangan Investigasi kebenaran laporan
👨‍💼 Governor Proses & tugaskan Memproses pengaduan, upload bukti
⚙️ Admin Kelola semua Verifikasi & publikasi

🔐 Authentication

POST /auth/register Public 
curl -X POST http://localhost:8080/api/auth/register \
  -H "Content-Type: application/json" \
  -d '{"username":"warga1","password":"123456","email":"warga@email.com","fullname":"Warga Satu","province_id":1}'
POST /auth/login Public 
curl -X POST http://localhost:8080/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username":"warga1","password":"123456"}'
POST /auth/google Public 
curl -X POST http://localhost:8080/api/auth/google \
  -H "Content-Type: application/json" \
  -d '{"google_id":"xxx","email":"user@gmail.com","fullname":"User Google","province_id":1}'

📝 Pengaduan (User/Public)

GET/complaints/categoriesPublic 
curl http://localhost:8080/api/complaints/categories
GET/complaints/status/{tracking_code}Public 
curl http://localhost:8080/api/complaints/status/JKT001
POST/complaints/submitNeed Auth 
curl -X POST http://localhost:8080/api/complaints/submit \
  -H "Authorization: Bearer TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"province_id":1,"location_detail":"Jl. Sudirman No.123","category_id":1,"description":"Jalan rusak parah"}'
GET/complaints/myNeed Auth 
curl -X GET http://localhost:8080/api/complaints/my \
  -H "Authorization: Bearer TOKEN"
